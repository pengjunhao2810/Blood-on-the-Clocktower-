# Blood on the Clocktower

血染钟楼 AI 社交推理游戏 — 包含 V1 经典版和 V2 增强版。

## 版本

| 版本 | 目录 | 说明 |
|------|------|------|
| V1 | [Blood on the Clocktower V1](./Blood%20on%20the%20Clocktower%20V1/) | 经典 AI 自走棋版 — Flask Web UI、Trouble Brewing 规则引擎、Agent 系统 |
| V2 | [Blood on the Clocktower V2](./Blood%20on%20the%20Clocktower%20V2/) | 增强版 — 角色思维引擎、人格系统、5 标签页私聊界面、ML 策略训练 |

## V2 新增功能

- **角色思维引擎** — 10 个角色专属思维模型，每轮对话自适应
- **人格系统** — 4 种性格影响发言风格 (冲动/冷静/话痨/新手)
- **5 标签页私聊界面** — 私聊气泡 + 说书人信息 + 夜晚顺序 + 角色能力 + 提名记录
- **邪恶方 ML 策略** — REINFORCE 策略梯度训练，控制杀目标/提名/投票
- **13MB 对话数据集** — 49 类模板，16000+ 条训练对话
